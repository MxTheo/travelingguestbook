Over mij

Mijn naam is Theo. Ik was klaar met studeren en begon te forenzen. Ik daagde mijzelf uit in het openbaar vervoer om kletspraatjes aan te gaan. Ik had behoefte aan andere persoonlijkere gesprekken, dan te praten over werk. Daarbij zat ik te experimenteren hoe ik dat kon doen. Ik stelde directe, persoonlijke vragen. Ik gaf dilemma's. Soms werden de experimenten gewaardeerd. Vaak ook niet. Dit was een paar jaar geleden. Toen zocht ik al naar manieren om het gesprek aan te gaan.

Tegenwoordig maak ik het wat makkelijker voor mezelf. Ik zeg hallo en stel mijn naam voor. Dat is meestal al genoeg gespreksstarter. Ik kan vragen hoe iemands dag is. Het hoeft niet lastig te zijn.

Het idee van deze applicatie sudderde dus al meerdere jaren in mijn onbewuste. Ik forens niet meer. Ik heb veel vrije tijd. Deze tijd besteed ik vooral aan vrijwilligerswerk. Ik heb mij voorgenomen om zoveel mogelijk in dienst te zijn voor anderen. Zolang ik spaargeld heb, kan ik dit volhouden. Ik hoop ook met deze applicatie iets te kunnen betekenen voor anderen.

Meer over mijn persoonlijke verhaal wil ik je graag verwijzen naar mijn substack. Dit over mij stukje is beperkt tot het ontwikkelen van de applicatie.

Bijdragen:

Zie github voor de broncode en de issues.

Naast programmeren aan deze applicatie, besteed ik veel tijd aan vrijwilligerswerk. Daardoor werk ik aan de applicatie zo af en toe. De ontwikkeling gaat zo langzaam, maar hopelijk wel stabiel.

Kwaliteit van code vind ik belangrijk. Ik streef mijn code leesbaar te houden, met documentatie. Ik streef een 100% test coverage. Zodat wat ontwikkeld is ook werkt. Ik maak nog steeds fouten. Zeker omdat ik het vooral in mijn eentje doe, waardoor mijn programmeren nog foutgevoeliger is. Ik heb wel wat programmeer ervaring, maar ik leer ook nog steeds veel terwijl ik aan het programmeren ben. Ik kan feedback dus goed gebruiken.
