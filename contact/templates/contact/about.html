You can start that conversation!

Doing that, you can empower others to start conversations

Ga dat kletspraatje aan, met de hulp van Traveling Guestbook. Traveling Guestbook is een spel die jouw aanknopingspunten biedt.

Zo kun je met iemand in de trein een kletspraatje houden. Je hoeft alleen de sociable door te geven.
Of misschien ben jij wel degene die aangesproken is voor een kletspraatje. Dan heb jij nu de sociable met de code in handen. Via Traveling Guestbook kun je dan een bericht achterlaten op de pagina van de sociable.

Sociables zijn de gespreksstarters. Het zijn fysieke dingen die van persoon tot persoon worden doorgegeven. Deze dingen kan van alles zijn, zoals een kaartje of een klein notitieboekje.

Op de sociable staat een code. Daarmee kun je de sociable terugvinden op de website. Elke keer dat het wordt doorgegeven, kan degene die de sociable heeft gekregen een berichtje plaatsen op de pagina van de sociable.

Sociables hebben verzoeken. Dit kan bijvoorbeeld een vraag zijn om iets over jezelf te delen. Of het kan de vraag zijn om iets in een boekje te tekenen. Voel je vrij en creatief om nieuwe verzoeken voor jou en andere gebruikers te bedenken.

Over mij:

Mijn naam is Theo. Ik was klaar met studeren en begon te forenzen. Ik daagde mijzelf uit in het openbaar vervoer om kletspraatjes aan te gaan. Ik had behoefte aan andere persoonlijkere gesprekken, dan te praten over werk. Daarbij zat ik te experimenteren hoe ik dat kon doen. Ik stelde directe, persoonlijke vragen. Ik gaf dilemma's. Soms werden de experimenten gewaardeerd. Vaak ook niet. Dit was een paar jaar geleden. Toen zocht ik al naar manieren om het gesprek aan te gaan.

Tegenwoordig maak ik het wat makkelijker voor mezelf. Ik zeg hallo en stel mijn naam voor. Dat is meestal al genoeg gespreksstarter. Het hoeft niet lastig te zijn.

Het idee van Traveling Guestbook sudderde dus al meerdere jaren in mijn onbewuste. Ik forens niet meer. Ik heb veel vrije tijd. Echter, veel van die vrije tijd gebruik ik voor vrijwilligerswerk. Ik heb mij voorgenomen om zoveel mogelijk in dienst te zijn voor anderen. Zolang ik spaargeld heb, kan ik dit volhouden. Ik hoop ook met Traveling Guestbook iets te kunnen betekenen voor anderen.

Bijdragen:

Zie github voor de broncode en de issues.

Ik besteed dus ook veel tijd aan vrijwilligerswerk. Daardoor werk ik aan de applicatie zo af en toe. De ontwikkeling gaat zo langzaam, maar hopelijk wel stabiel.

Kwaliteit van code vind ik belangrijk. Ik streef mijn code leesbaar te houden, met documentatie. Ik streef een 100% test coverage. Zodat wat ontwikkeld is ook werkt. Ik maak nog steeds fouten. Zeker omdat ik het vooral in mijn eentje doe, waardoor mijn programmeren nog foutgevoeliger is.
