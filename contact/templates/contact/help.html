Gesprekseindiger

Deze applicatie is bedoeld als gesprekseindiger. Via deze website kun je codes aanmaken, die je vervolgens op voorwerpen kunt schrijven. Met de code kun je iemand bedanken voor het gesprek.
Er zijn vele manieren om gesprekken te openen. Uiteindelijk kun je alleen zelf het gesprek openen. Geen gespreksstarter op de wereld kan dat voor je doen. Met de code kun je wel iemand bedanken voor het gesprek.

Pay it forward

Dit is belangrijk. Dat de code wordt doorgegeven. Als ieder die bedankt wordt, anderen ook bedankt met dezelfde code, dan kun je zo de reis volgen van de code. De code reist van persoon tot persoon, elke keer via een nieuw, uniek gesprek. Bedankt voor het doorgeven!

Deel je ervaringen!

Je hebt een code gekregen! Met die code kom je bij de pagina waar je een berichtje achter kunt laten. Deel je ervaringen van het gesprek. Zo kunnen anderen zien dat de code, die zij nu hebben, ook bij jou langs is geweest. Zo ontmoeten we elkaar, omdat anderen werkelijk de moeite hebben gedaan om een gesprek aan te knopen. Lees de ervaringen van alle andere mensen die ook de moeite hebben gedaan om het kletspraatje aan te gaan. Via ieder die het gesprek is aangegaan zijn we met elkaar verbonden.

Start zelf!

Met de code kan iedereen een berichtje achter laten. Daar hoef je geen account voor aan te maken. Alleen gebruikers kunnen codes aanmaken. Zij kunnen die gespreksketen starten. Zet je in voor een sociale wereld en start zelf met gespreksketens! Maak een account aan.
