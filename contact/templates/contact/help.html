Sociables, and Messages

Sociable

Het fysieke item dat wordt doorgegeven van persoon tot persoon. Dit kan van alles zijn, zoals een kaartje. Op dit fysieke item staat een code, waarmee je de pagina van de sociable kan bereiken.

Message

Het bericht dat de ontvanger van de sociable achterlaat op de pagina van de sociable. Met de code op de sociable kom je op de pagina waar je een berichtje kan achterlaten.
