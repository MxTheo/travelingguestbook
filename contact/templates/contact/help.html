Gesprekseindiger

Deze applicatie is bedoeld als gesprekseindiger. Dankers zijn voorwerpen met een code erop. Met danker kun je iemand bedanken voor het gesprek.
Er zijn vele manieren om gesprekken te openen. Uiteindelijk kun je alleen zelf het gesprek openen. Geen gespreksstarter op de wereld kan dat voor je doen. Met een danker kun je wel iemand bedanken voor het gesprek.

Pay it forward

Dit is belangrijk. Dat de danker wordt doorgegeven. Als ieder die bedankt wordt, anderen ook bedankt met dezelfde danker, dan kun je zo de reis volgen van de danker. De danker reist van persoon tot persoon, elke keer via een nieuw, uniek gesprek. Bedankt voor het doorgeven!

Deel je ervaringen!

Op de danker staat een code. Met die code kom je bij de pagina waar je een berichtje achter kunt laten. Deel je ervaringen van het gesprek. Zo kunnen anderen zien dat de danker, die zij nu hebben, ook bij jou is langsgeweest. Zo ontmoeten we elkaar, omdat anderen werkelijk de moeite hebben gedaan om een gesprek aan te knopen. Lees de ervaringen van alle andere mensen die ook de moeite hebben gedaan om het kletspraatje aan te gaan. Via dankers zijn we met elkaar verbonden.

Start zelf!

Met de code kan iedereen een berichtje achter laten. Daar hoef je geen account voor aan te maken. Alleen gebruikers kunnen dankers aanmaken. Zij kunnen die gespreksketen starten. Zet je in voor een sociale wereld en start zelf met gespreksketens! Maak een account aan.
